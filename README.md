# bus_ticket_booking_app_csharp_console

This this a demo Bus Ticket Booking Application on C# console.

Features:
  i) Create User
  ii) Show User
  iii) Create Bus
  iv) Show Buses
  v) Create Schedule
  vi) Show Schedules
  vii) Show Schedule Details
  viii) Book Ticket
  ix) Show Invoices
  x) Payment
  xi) Show booked ticket
